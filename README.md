dvl-core [![version]](http://rubygems.org/gems/dvl-core) [![build_status]](https://circleci.com/gh/dobtco/dvl-core)
====

Base styles for the DOBT View Layer.

## Usage

### Gemfile

```
gem 'dvl-core'
```

### assets/javascripts/application.css.scss

```
@import 'dvl/core';
```
